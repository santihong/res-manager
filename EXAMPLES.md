# 使用示例

## 场景一：下载网页中的所有产品图片

**适用场景**：电商网站、图片展示网站

1. 打开目标网页（如：淘宝商品页、京东商品页）
2. 点击插件图标
3. 在「当前页面」标签下：
   - ✅ 勾选 JPG、PNG、WebP
   - 设置最小尺寸：`300`（过滤小图标）
4. 点击「下载当前页面图片」
5. 所有产品大图会自动下载

## 场景二：批量下载壁纸

**适用场景**：壁纸网站、摄影网站

1. 打开壁纸网站（如：Unsplash、Pexels）
2. 点击插件图标
3. 设置：
   - ✅ 勾选所有格式
   - 最小尺寸：`1920`（只下载高清大图）
4. 点击下载
5. 获得高质量壁纸

## 场景三：下载指定网页的图片（无需打开）

**适用场景**：已知URL，想快速获取图片

1. 点击插件图标
2. 切换到「指定URL」标签
3. 输入网址：`https://example.com/gallery`
4. 选择格式和尺寸
5. 点击「下载指定页面图片」
6. 插件会自动：
   - 后台打开该页面
   - 提取所有图片
   - 下载完成后关闭页面

## 场景四：下载网站图标和Logo

**适用场景**：收集品牌Logo、网站图标

1. 打开目标网站首页
2. 点击插件图标
3. 设置：
   - ✅ 勾选 PNG、SVG（Logo常用格式）
   - 最小尺寸：`0`（包含所有尺寸）
4. 下载后筛选需要的Logo

## 场景五：下载文章配图

**适用场景**：博客、新闻网站

1. 打开文章页面
2. 点击插件图标
3. 设置：
   - ✅ JPG、PNG
   - 最小尺寸：`500`（只要配图，不要图标）
4. 获得文章中的所有配图

## 高级技巧

### 技巧1：精确控制下载内容

```
最小尺寸设置建议：
- 0px：下载所有图片（包括图标）
- 100px：过滤掉小图标
- 300px：只要中等以上的图片
- 1000px：只要大图/高清图
- 1920px：只要超高清壁纸级别
```

### 技巧2：格式选择策略

```
照片类网站：JPG + WebP
设计类网站：PNG + SVG
动图网站：GIF
全能模式：全选
```

### 技巧3：批量下载多个页面

虽然插件每次只能下载一个页面，但可以：
1. 打开多个标签页
2. 依次在每个标签页点击插件下载
3. 所有图片会下载到同一个文件夹

### 技巧4：避免重复下载

插件会自动为重名文件添加序号：
- `image.jpg`
- `image(1).jpg`
- `image(2).jpg`

### 技巧5：快速测试

想知道页面有多少图片？
1. 点击插件
2. 选择格式
3. 点击下载
4. 查看「找到图片」数量
5. 如果不想下载，关闭插件窗口即可

## 注意事项

⚠️ **版权提醒**：下载的图片可能受版权保护，请遵守相关法律法规，仅用于个人学习和研究。

⚠️ **网站限制**：某些网站可能有反爬虫机制，如果下载失败，这是正常现象。

⚠️ **下载数量**：如果页面图片过多（100+），下载可能需要几分钟时间，请耐心等待。

## 常见问题

**Q: 为什么有些图片下载后打不开？**  
A: 可能是图片URL已失效或格式不标准，这种情况较少见。

**Q: 可以下载视频吗？**  
A: 不可以，本插件仅支持图片格式。

**Q: 下载的图片质量会压缩吗？**  
A: 不会，下载的是原图，保持原始质量。

**Q: 可以选择下载位置吗？**  
A: 图片会下载到浏览器默认下载目录的 `images` 子文件夹，可以在浏览器设置中修改默认下载位置。
