# 图标文件说明

## 📌 重要提示

在使用插件之前，你需要先生成图标文件！

## 🎨 如何生成图标

### 方法一：使用图标生成器（推荐）

1. 用浏览器打开项目根目录下的 `icon-generator.html` 文件
2. 页面会自动生成三个尺寸的图标
3. 点击「下载所有图标」按钮
4. 将下载的三个文件放到这个 `icons` 文件夹中：
   - icon16.png
   - icon48.png
   - icon128.png

### 方法二：使用自己的图标

如果你想使用自己设计的图标：

1. 准备三个尺寸的PNG图片：
   - 16x16 像素（工具栏显示）
   - 48x48 像素（扩展管理页显示）
   - 128x128 像素（Chrome商店显示）

2. 命名为：
   - icon16.png
   - icon48.png
   - icon128.png

3. 放到这个文件夹中

## ✅ 验证图标

图标文件应该是：
- 格式：PNG
- 背景：可以是透明或纯色
- 内容：建议使用图片相关的图标设计

## 🚫 常见问题

**Q: 没有图标可以安装插件吗？**  
A: 可以安装，但插件图标会显示为默认的灰色图标，不美观。

**Q: 图标必须是这些尺寸吗？**  
A: 是的，Chrome要求这三个标准尺寸。

**Q: 可以使用JPG格式吗？**  
A: 不可以，必须是PNG格式。

---

**提示**：使用 `icon-generator.html` 是最简单的方法，只需一键即可生成所有图标！
