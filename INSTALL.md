# 快速安装指南

## 📦 安装步骤

### 第一步：生成图标

1. 用浏览器打开 `icon-generator.html` 文件
2. 点击「下载所有图标」按钮
3. 将下载的三个图标文件（icon16.png、icon48.png、icon128.png）移动到 `icons` 文件夹中

### 第二步：安装插件

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的「开发者模式」开关
4. 点击「加载已解压的扩展程序」
5. 选择 `res-manager` 文件夹
6. 完成！

## ✅ 验证安装

安装成功后，你会看到：
- 浏览器工具栏出现插件图标（紫色渐变的图片图标）
- 在 `chrome://extensions/` 页面看到「图片资源下载器」

## 🚀 开始使用

1. 打开任意包含图片的网页（如：百度图片、Pinterest等）
2. 点击插件图标
3. 选择图片格式
4. 点击「下载当前页面图片」
5. 图片会自动下载到 `下载/images/` 文件夹

## 🔧 常见问题

**Q: 找不到「开发者模式」开关？**  
A: 在 `chrome://extensions/` 页面的右上角，是一个蓝色的开关按钮。

**Q: 提示「无法加载扩展程序」？**  
A: 确保选择的是 `res-manager` 文件夹本身，而不是其父文件夹。

**Q: 图标显示为灰色？**  
A: 确保已经生成并放置了图标文件到 `icons` 文件夹中。

**Q: 点击插件没反应？**  
A: 刷新页面后再试，或者重新加载插件。
